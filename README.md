# Entertainment Guild 🎮🎬🎵

A full-stack React e-commerce web application for entertainment 
products including movies, games, and music.

## Overview
Entertainment Guild is an online store where users can browse, 
purchase, and manage entertainment products across multiple 
categories. It features a full checkout flow, user authentication, 
and an admin dashboard.

## Features
- 🛍️ Browse products by genre and category
- 🔍 Product description and detail pages
- 🛒 Shopping cart with quantity management
- 💳 Full checkout flow with address and payment forms
- 🔐 User sign in and sign up
- 👤 User profile and order history
- 🔒 Protected admin dashboard for store management
- 📱 Responsive design for mobile and desktop
- 🌙 Light and dark mode support

## Tech Stack
React · JavaScript · Material UI (MUI) · React Router · 
Context API · Custom Hooks

## Project Structure
- `Components/` — Main app pages and UI components
- `Admin/` — Admin dashboard components
- `CheckOutComponents/` — Checkout flow (address, payment, review)
- `Context/` — Global cart and session state management
- `Hooks/` — Custom React hooks (form validation, session)
- `Api/` — API call handlers
- `Data/` — Static data and product data
- `Helpers/` — Utility helper functions
- `UI/` — Reusable UI components
